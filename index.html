Crystal Collector Game

How to Play?

Game Mechanics

At the start of the game you will be given a random number of crystals to collect.

There are four crystals below. By clicking on a crystal you will add a specific number to your collection.

You win the game by matching the total number of crystals you collected to what is needed, go over the number of needed crystals and you lose.

The value of each crystal is hidden from you until you click on it.

Each time the game starts, the values of each crystal below changes.
